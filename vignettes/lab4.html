General information

Package implements linear regression models, called linreg and is called as an example:

ex_form <- Petal.Length ~ Sepal.Width + Sepal.Length
linreg_ex <- linreg$new(formula=ex_form, data=iris)

The package calculates the following formulas: Regression coefficients: \[\mathbf{\hat{\beta} = \left(X^\intercal X\right)^{-1} X^\intercal y}\] The fitted values: \[\mathbf{\hat{y} = X \hat{\mathrm{\beta}}}\] The residuals: \[\mathbf{\hat{e} = y - \hat{{y}} = y - X \hat{\mathrm{\beta}}}\] The degrees of freedom: where \(n\) is the number of observations and \(p\) is the number of parameters in the model. \[df=n-p\]

The residual variance: \[\hat{\sigma}^2 = \mathbf{\frac{e^\intercal e}{df}}\] The variance of the regression coefficients: \[\mathbf{\mathrm{Var} \left( \hat{\mathrm{\beta}} \right)} = \hat{\sigma}^2 \mathbf{(X^\intercal X)^{-1}}\] The t-values for each coefficient: \[ t_\beta = \frac{\hat{\beta}}{\sqrt{\mathrm{Var} \left( \hat{\beta} \right)}}\]
